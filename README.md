## 🛠️ Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Deployment**: Vercel + Cloudflare Pages
- **Icons**: Lucide React
- **TypeScript**: Full type safety

## 🚀 Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## 📦 Deployment

### Vercel
Auto-deploys from `master` branch to https://autotrip-five.vercel.app

### Cloudflare Pages
Auto-deploys from `master` branch to https://autotrip-perth.pages.dev
